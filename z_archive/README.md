# Archived `cloud-data-{name}.json` files

These 22 files are **deprecated**. They were the historical per-purpose split
files of the cloud-data shared registry.

## What replaced them

The new master file is **`_cloud-data-consolidated.json`** at the repo root —
a single document that contains every slice these split files used to carry.
Consumers now read consolidated and extract the relevant nested key:

| deprecated file | nested key in `_cloud-data-consolidated.json` |
|---|---|
| `cloud-data-topology.json` | top-level (consolidated *is* a superset of topology) |
| `cloud-data-configs.json` | `.configs` |
| `cloud-data-deps.json` | `.deps` |
| `cloud-data-home-manager.json` | `.home_manager` / `._home_manager` |
| `cloud-data-firewall-rules.json` | `.firewalls` |
| `cloud-data-gha-config.json` | `._gha` |
| `cloud-data-cloudflare-dns.json` | `.dns.cloudflare` |
| `cloud-data-monitoring-targets.json` | derived from `.services[*].monitoring` |
| `cloud-data-backup-targets.json` | derived from `.services[*].backup` |
| `cloud-data-secrets-env-var-names.json` | derived from `.services[*].secret_env_vars` |
| ... (other splits) ... | derived from `.services[*]` / `.vms[*]` |

## Why kept (not deleted)

Soft transition. Some out-of-tree consumers may still clone this repo and
read the deprecated paths. Files in `z_archive/` keep them discoverable as
historical artefacts; new consumers should never reference them.

## Source of truth

These files are now generated by the cloud repo's derive engine and pushed
here by the `bc-obs_dagu/sync_cloud-data.yaml` DAG (currently disabled —
`sync_cloud-data.yaml.disabled`).

Pipeline (in the cloud repo):
- consolidator: `2_configs/src/engines/cloud-data-config-consolidated.ts`
- derive: `2_configs/src/engines/cloud-data-config-derive.ts`
- master: `2_configs/dist/_cloud-data-consolidated.json`
- per-container: `2_configs/dist/build-{name}.json`
- archived (here too): `2_configs/dist/z_archive/cloud-data-{name}.json`

Rebuild: `bash 2_configs/build.sh all` (in the cloud repo).
